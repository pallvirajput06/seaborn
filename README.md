# Seaborn Practice 🐍📊

This repository contains my hands-on practice with **Seaborn**, a powerful Python library for statistical data visualization. It includes examples of distribution plots, categorical plots, relational plots, regression plots, heatmaps, pair plots, and customization techniques.

### Topics Covered

* Distribution Plots (`histplot`, `displot`, `kdeplot`)
* Categorical Plots (`barplot`, `countplot`, `boxplot`, `violinplot`)
* Relational Plots (`scatterplot`, `lineplot`)
* Regression Plots (`regplot`, `lmplot`)
* Pair Plot & Joint Plot
* Heatmaps and Correlation Analysis
* Plot Customization (colors, styles, labels, titles)

### Tech Stack

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn

This repository is part of my Data Analytics and Data Science learning journey, where I practice visualization techniques and build a strong foundation in exploratory data analysis (EDA).
